# Arcana 

A card game inspired by **Magic The Gathering** and **Yu-Gi-Oh!**, developed in **Godot Engine**.  
The game focuses on strategic dueling, summoning creatures, casting spells, and building decks.

## About
Arcana is a fantasy card game where players build decks, summon creatures, and battle opponents using strategy and skill.  
Inspired by popular trading card games, this project is being developed as a Godot-based game with custom rules and mechanics.

## Features
- Card system with customizable attributes (mana, attack, health, special abilities)  
- Turn-based gameplay  
- Placeholder art and UI (development stage)  
- Deck management and card interaction  

## Current Status
- Project created in **Godot 4.x**  
- Basic project structure set up  
- Placeholder scenes for main menu and card board  
- No gameplay implemented yet
